📍 viacepPoke

Projeto frontend simples que integra duas APIs — ViaCEP para consultar dados de endereço pelo CEP e PokéAPI (opcional) para trazer informações de Pokémon — usando fetch no JavaScript.

⚡ Esse projeto tem como objetivo exercitar o consumo de APIs públicas e manipulação assíncrona de dados no navegador.

🧪 Funcionalidades

✔️ Consulta de CEP com preenchimento automático de endereço
✔️ Validação simples do CEP informado
✔️ Integração com APIS públicas
✔️ Exibição de dados no DOM com JavaScript
✔️ Experiência prática com fetch e async/await


🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript (fetch API e async/await)

API pública ViaCEP (consulta de CEP)

(Opcional) PokéAPI — para dados de Pokémon

🚀 Como usar

Clone o repositório

git clone https://github.com/NicolleMonteiro/viacepPoke.git

Abra o arquivo

Abra o index.html no seu navegador (duplo clique ou use Live Server)

Digite um CEP válido

Insira um CEP no campo e veja os dados serem preenchidos automaticamente.

📌 Exemplos de API

Consulta ViaCEP:

https://viacep.com.br/ws/01001000/json/

Retorna endereço completo com logradouro, bairro, cidade e estado.
